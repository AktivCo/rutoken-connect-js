# Пример использования модуля Рутокен Коннект

Собранный пример можно посмотреть по [ссылке](https://aktivco.github.io/rutoken-connect-js/).

## Сборка

### npm

```sh
npm install
npm run build
```

### yarn

```sh
yarn
yarn build
```

Собранный пример окажется в директории dist.
